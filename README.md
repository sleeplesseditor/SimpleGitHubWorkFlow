## Description
A simple example of the use of Github workflows, using a basic React app, as part of coursework for [Maximilian Schwarzmüller's Udemy course](https://www.udemy.com/course/github-actions-the-complete-guide).

## Changes from Original Project
- Used a Vite React app with TypeScript template to recreate the coursework example app
- Adapted some of the workflow yaml files to use slightly different patterns (i.e. text printing)
