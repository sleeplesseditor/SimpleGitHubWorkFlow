## Description
A simple example of the use of Github workflows with a basic React app, as part of coursework for [Maximilian Schwarzmüller's Udemy course](https://www.udemy.com/course/github-actions-the-complete-guide).

## Changes from Original Coursework
- Used a Vite React app with TypeScript template to recreate the coursework example app
- Adapted some of the workflow yaml files to use slightly different patterns (i.e. text printing)
    - Updated use of v3 actions to v4 where relevant
    - Updated use of Node versions to more contemporary ones (i.e. 24)

## Potential Further Improvements
- Break out workflows to separate branches
- Consider storage of actions in separate repo

## Additional Notes
To see the main focus of this repo's work, look at the 'Actions' tab at the top of the GitHub repo page (as well as viewing the yaml files in the .github folder)
